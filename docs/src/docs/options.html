Options

You define the behaviour of your component by passing options to as the second argument to the `skate()` function.

The following options are supported:

  • type
  • extends
  • created
  • attached
  • detached
  • attribute
  • template
  • events
  • properties
  • prototype

type

Specifies the type of binding that this component supports using the element type constants.

extends

The name of the element that this component is extending. This is synonymous with the W3C Custom Element extends option.

When you specify the extends option, instead of your component name mapping to the tagName of an element, it maps to the value of the is attribute. This conforms with the spec.

However, if you use this option in conjunction with the type option, Skate will only bind the component to an element that has a tag name matching the extends option.

created

Called when the element is created as defined in the Element Lifecycle.

attached

Called when the element is attached as defined in the Element Lifecycle.

detached

Called when the element is detached as defined in the Element Lifecycle.
